# LanguageDetectionTool
# Introduction
Language Detection Tool is a machine learning-based application that automatically identifies the language of a given text input. The system uses Natural Language Processing (NLP) techniques and machine learning algorithms to detect multiple world languages accurately. It helps applications process multilingual content efficiently and improves communication in systems like chatbots, translation tools, and customer support platforms.
# Features
Automatic language detection
Supports multiple languages
Fast and accurate prediction
User-friendly web interface
Machine learning-based classification
Real-time text processing
Lightweight and scalable system
# Technologies Used
Python
Flask
HTML5
CSS3
Jinja2
langdetect Library
TF-IDF Vectorizer
Naive Bayes Classifier
Visual Studio Code (VS Code)
# Setup Instructions
Install Python on your system
Install required libraries using:
pip install flask langdetect scikit-learn
Open the project in Visual Studio Code
Run the application using:
python app.py
Open the browser and visit:
http://127.0.0.1:5000/
